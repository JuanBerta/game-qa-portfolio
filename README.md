## 📋 Standard Bug Report Format
Every bug report in this repository follows a standardized structure:
1. **Summary & Severity Rating**
2. **Environment / Build Version**
3. **Preconditions & Reproduction Steps**
4. **Expected vs. Actual Result**
5. **Visual Proof / Logs / Attachments**
